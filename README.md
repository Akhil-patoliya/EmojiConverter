# EmojiConverter
This is an Android library used to convert realtime text to Emojis in edittext. 
## Usage
Add below code in build.gradle
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
Add a dependency
```
dependencies {
	        compile 'com.github.AnkitKiet:EmojiConverter:1.00'
	}
```
